# From AI Hype to Platform Reality

A comprehensive demo showcasing production-ready AI service architecture using Red Hat OpenShift, Red Hat OpenShift AI, Red Hat Developer Hub (Backstage), OpenShift GitOps (Argo CD), AMQ Streams (Kafka), and modern cloud-native technologies.

## Overview

This repository consolidates the complete Red Hat One demo, featuring:

- **AI-enabled microservices** working together in a realistic customer service scenario
- **Kafka topics** for event-driven communication
- **Red Hat Developer Hub (Backstage)** for developer experience and service catalog
- **OpenShift GitOps (ArgoCD)** for automated deployment and lifecycle management
- **AMQ Streams (Kafka)** for reliable message streaming
- **Milvus Vector Database** for semantic search and RAG capabilities
- **LangChain-based agents** for intelligent message routing and analysis
- **GitHub Actions** for image builds and CI
